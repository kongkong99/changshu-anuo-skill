## 术语生成公式

```
[具体事物] + [矛盾/夸张] + [自我中心] + [命理归因]

例：
- 深蹲200公斤 + 十年前=现在300公斤 + 我能蹲 + 这也是命注定
- 左旋果冻 + 可以当果冻吃 + 我一直在吃 + 效果有那么一点点
```
